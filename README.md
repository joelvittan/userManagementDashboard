# User Management System

## Project Overview

This User Management System is designed to streamline the process of managing user data within an organization. It features a responsive web interface that allows administrators to view, add, edit, and delete user details efficiently. The system is built with modern web technologies for scalability and ease of maintenance.

## Features

- **View Users**: Displays a list of users with pagination support.
- **Add Users**: Provides a form interface for adding new users.
- **Edit Users**: Allows editing existing user information via a detailed form.
- **Delete Users**: Supports user deletion with confirmation prompts.
- **Responsive Design**: Fully responsive on both desktop and mobile devices.

## Technologies Used

- **Next.js**: A React framework for server-side rendering and generating static websites.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Axios**: A promise-based HTTP client for browser and Node.js.
- **Material-UI**: React components for faster and easier web development following Material Design.
- **Git**: Used for version control to track changes and collaborate effectively.

## Usage

* **Main Interface** : The homepage lists all users. Use pagination to navigate through different pages.
* **Add a User** : Click the 'Add User' button to open a form. Fill and submit it to add a new user.
* **Edit a User** : Click 'Edit' next to any user, modify the details in the form, and submit.
* **Delete a User** : Click 'Delete' next to any user, then confirm in the prompt to remove the user.
